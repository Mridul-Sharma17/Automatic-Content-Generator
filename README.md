## 🛠️ Troubleshooting

### Common Issues & Solutions

**"Module not found" error**:
```bash
source ai_video_env/bin/activate
pip install -r requirements.txt
```

**Video generation slow**:
- Normal for first video (downloads AI models)
- Subsequent videos much faster

**Want different voice**:
- Edit `ai_video_generator.py`
- Change voice in `text_to_speech()` function
- Available voices: "en-US-AriaNeural", "en-US-JennyNeural", "en-GB-SoniaNeural"
